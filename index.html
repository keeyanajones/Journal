About this Code Journal

Last updated on March 3rd, 2020

This Github Repository is a series of in progress writings about my work. It covers Java, JavaScript, Python, R, SQL, and typical language sub-disciplines related to Computer Science, Data Science, Artificial Intelligence, Cloud Computing, Game, and Application Development.


What works best for me when learning how to implement any new technology is to keep a hand written journal, to keep things I am working on hand so I can write down random thoughts, my success logs, things to revisit later, goals, and lessons learned. I try to read through and recreate the pages of analysis, parsing and formatting text, math, data structures, code snippets, diagrams, programming paradigms, dynamic programming, and geometry in this repository.
