twitch-video-overlays

An overlay can be a border around your webcam, or it can be a full image thing taking up the whole frame. It's basically a graphic (normally a transparent PNG file) that sits above your game capture window, and your webcam.

Installation

Fire up a console and type:


apm install atom-color-ui

Or, inside Atom's settings select Install and then search for this package.

Configuration

Atom Color UI supports different accent colors. To change it, go to Settings > Themes and click the cog icon next to the theme selector. You'll find the color picker there.

Screenshots

Here's the obligatory screenshot. With Atom Color Syntax


apm install atom-color-syntax

With Light syntax theme variant


apm install atom-color-syntax-light

With Dark syntax theme variant


apm install atom-color-syntax-dark
