A dynamic Plugin for Netbeans that follows Color Theory Design Guidelines. Best with Netbeans 11 or later. Inspired by Keeyana.

Installation

Fire up a console and type:


apm install Netbeans Plugin

Or, inside Netbeans Plugin settings select Install and then search for this package.

Configuration

Netbeans Plugin supports different accent colors. To change it, go to Settings > Plugin and click the cog icon next to the theme selector. You'll find the color picker there.
